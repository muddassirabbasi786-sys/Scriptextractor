# YouTube Script Extractor - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep charcoal (#1a1a1a) for main backgrounds
- **Secondary**: Warm white (#fafafa) for content areas
- **Accent**: Electric blue (#0066ff) for interactive elements
- **Success**: Forest green (#22c55e) for completion states
- **Warning**: Amber (#f59e0b) for processing states
- **Text**: High contrast dark gray (#111827) on light, light gray (#f3f4f6) on dark

### Typography
- **Display Font**: "Tiempos Headline" - Bold serif for impactful headings
- **Body Font**: "Suisse Int'l" - Clean sans-serif for readability
- **Monospace**: "JetBrains Mono" - For code/transcript display
- **Font Sizes**: Large headings (48px+), medium subheadings (24px), small body text (16px)

### Visual Language
- **Minimalist Efficiency**: Clean lines, generous whitespace, purposeful elements
- **Professional Precision**: Sharp edges, consistent spacing, systematic approach
- **Subtle Sophistication**: Refined details, quality materials, understated elegance
- **Interactive Clarity**: Clear affordances, immediate feedback, intuitive flow

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and state transitions
- **ECharts.js**: Data visualization for usage statistics
- **p5.js**: Creative coding for background particle effects
- **Splide.js**: Smooth carousel for feature demonstrations

### Animation & Effects
- **Particle Background**: Subtle floating particles using p5.js for depth
- **Morphing Buttons**: Smooth color and shape transitions on hover
- **Progress Animations**: Fluid progress bars during extraction
- **Text Reveal**: Staggered character animations for headings
- **Card Hover**: 3D tilt effects on feature cards

### Header Effect
- **Gradient Flow**: Animated gradient background with subtle movement
- **Floating Elements**: Geometric shapes that respond to scroll
- **Depth Layers**: Multiple background layers for visual richness

### Layout Principles
- **Grid System**: 12-column responsive grid with consistent gutters
- **Vertical Rhythm**: 24px baseline grid for consistent spacing
- **Content Hierarchy**: Clear visual hierarchy through size and contrast
- **Responsive Design**: Mobile-first approach with breakpoints at 768px, 1024px, 1440px

### Interactive Elements
- **Button States**: Hover, active, disabled states with smooth transitions
- **Form Validation**: Real-time feedback with color and icon changes
- **Loading States**: Skeleton screens and progress indicators
- **Success Feedback**: Checkmark animations and green highlights

### Background Treatment
- **Consistent Base**: Maintained dark charcoal throughout
- **Subtle Texture**: Very light noise texture for depth
- **Accent Gradients**: Blue accent gradients for section separation
- **No Section Colors**: Unified background with visual interest through other elements
